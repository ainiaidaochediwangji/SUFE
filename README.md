# SUFE 校园信息聚合工具

> 上海财经大学（SUFE）校园信息聚合工具集，支持课程资料下载、就业信息追踪、成绩查询、自动评教等功能。采用全异步架构，通过统一的 SSO 认证访问学校多个信息系统。

---

## 功能模块

| 模块 | 命令 | 说明 |
|------|------|------|
| **Canvas 课程下载** | `sufe canvas` | 批量下载 Canvas LMS 课程文件、模块文件、作业附件 |
| **就业信息追踪** | `sufe career` | 抓取就业网招聘信息，支持增量更新与变更检测 |
| **成绩查询** | `sufe grade` | 获取 EAMS 成绩数据，支持学期分析与 CSV 导出 |
| **自动评教** | `sufe evaluate` | 自动完成研究生课程评教（排序 + 问卷打分） |

### Canvas 课程下载

- SSO 自动登录（RSA 加密 + 滑块验证码破解）
- API + HTML 双模式获取课程列表
- 并发下载课程文件、模块文件、作业附件
- **增量更新检测**：基于文件大小 + `updated_at` 时间戳双重校验，文件内容更新时自动重新下载
- 智能去重合并，支持断点续传
- 为每个作业生成包含元数据的 HTML 页面

### 就业信息追踪

- 异步并发抓取招聘列表与详情
- 增量更新：缓存已抓取数据，仅刷新变更项
- 变更检测：自动识别新增、更新、删除的招聘信息
- 多格式导出：JSON / CSV / Excel

### 成绩查询

- 通过 EAMS 教务系统获取成绩数据
- 支持当前学期和历史成绩查询
- 自动计算加权平均绩点
- 按学期分组展示，每学期课程成绩排序
- 导出为 JSON 和 CSV 格式

### 自动评教

- 自动完成研究生课程评教
- 排序打分：为所有课程提交不重复的排名值
- 问卷打分：为每门课程的问卷自动填写评分
- 评分等级可在配置文件中调整（excellent / good / fair / bad / poor）

## 快速开始

### 1. 安装依赖

需要 Python 3.10+。第三方依赖统一在 `pyproject.toml` 的 `[project.dependencies]` 声明（`requirements.txt` 与之保持同步）：

```bash
pip install -e .
# 或
pip install -r requirements.txt
```

### 2. 配置账号

```bash
cp .env.example .env
```

编辑 `.env`：

```env
user=学号
pwd=密码
```

> `.env` 已加入 `.gitignore`，不会被提交。

### 3. 运行

```bash
# Canvas 课程下载
python -m sufe canvas

# 就业信息抓取
python -m sufe career

# 成绩查询与分析
python -m sufe grade

# 自动评教
python -m sufe evaluate

# 查看帮助
python -m sufe --help
```

### 4. 配置文件（推荐）

复制示例配置文件并按需修改：

```bash
cp sufe.yaml.example sufe.yaml
```

编辑 `sufe.yaml`：

```yaml
# ── Canvas 模块配置 ──
canvas:
  output_dir: course                    # 课程文件下载目录（相对于项目根目录）
  concurrency: 2                        # 并发下载课程数
  file_concurrency: 4                   # 课程文件并发下载数
  module_concurrency: 8                 # 模块文件发现并发数

# ── Career 就业信息模块配置 ──
career:
  output_dir: career_data/sxzpxx        # 就业数据输出目录（相对于项目根目录）
  concurrency: 8                        # 详情页抓取并发数
  page_concurrency: 3                   # 分页列表抓取并发数
  max_items: 300                        # 最大抓取条目数

# ── Grade 成绩查询模块配置 ──
grade:
  output_dir: grade_data                # 成绩数据输出目录（相对于项目根目录）
  export_formats:                       # 导出格式（支持 json / csv / excel）
    - json
    - csv
  generate_report: true                 # 是否生成分析报告（加权 GPA、学期分组等）

# ── Evaluation 评教模块配置 ──
evaluation:
  level: excellent                      # 问卷评分等级
                                        # excellent = 非常好
                                        # good      = 较好
                                        # fair      = 一般
                                        # bad       = 不好
                                        # poor      = 非常不好
  concurrency: 4                        # 并发评教课程数
```

**配置优先级：** CLI 参数 > `sufe.yaml` > `config.py` 默认值

> `sufe.yaml` 已加入 `.gitignore`，不会被提交到版本控制。

```
│
└── 学科代码/
    └── 课程名称B/
        ├── files/
        ├── modules/
        ├── assignments/
        └── _download_report.json
```

### 增量更新机制

Canvas 模块采用**双重校验**检测文件更新：

1. **文件大小对比**：本地文件大小与 API 返回的 `size` 字段对比
2. **更新时间对比**：读取上次的 `_download_report.json`，对比 API 返回的 `updated_at` 时间戳

| 场景 | 文件大小 | updated_at | 行为 |
|------|---------|------------|------|
| 文件不存在 | - | - | 下载 |
| 文件未变化 | 相同 | 相同 | 跳过 |
| 文件大小变化 | 不同 | 任意 | 重新下载 |
| 文件内容更新（大小不变） | 相同 | 不同 | 重新下载 |
| 文件内容更新（大小变化） | 不同 | 不同 | 重新下载 |

1. **网络环境**：需在校园网或 VPN 环境下访问学校系统
